# Sistema de Gerenciamento de Biblioteca (C)

Este projeto consiste em um sistema de gerenciamento de biblioteca desenvolvido em linguagem C, criado como parte de uma atividade avaliativa da faculdade.

O sistema permite o controle básico de livros, usuários e empréstimos, utilizando conceitos fundamentais de 
programação estruturada.

## Funcionalidades

- Cadastro de usuários
- Buscar usuários
- Cadastro de livros
- Buscar livros
- Emprestar livros
- Listar empréstimos ativos
- Renovar empréstimos
- Devolver livros
- Armazenamento e leitura de dados em arquivos .txt

## Tecnologias Utilizadas

- Compilador GCC
- Ambiente de desenvolvimento: Visual Studio Code
